Status

Delivered Features

- Canonical trick model for simple tricks, combos, and exits
- Assisted trick input in CLI and GUI
- Canonical trick storage in runtime/history/save-load/undo

Not Yet Implemented

Important Rule Boundary
- Uniqueness reads validated tricks only
- Repetition reads failed launched tricks only
- successful launches do not count for Repetition
- cancelled turns do not count for Repetition
