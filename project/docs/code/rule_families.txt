SkateGame Engine Prototype - Rule Families After V8

Objectif
Ce document cartographie les grandes familles de regles que le moteur supporte
deja ou devra absorber ensuite, en partant de l'architecture reelle apres V8.

Ce que V8 a change
- le moteur sait maintenant raisonner sur une identite canonique de trick
- `Uniqueness` et `Repetition` existent comme vraies regles fines runtime
- `dictionary/` introduit une famille nouvelle : les regles de validite d'un
  trick selon une discipline et un profil

Grandes familles actuelles

1. Regles de structure de match
- qui joue
- qui defend
- comment tourne la partie
- couche principale :
  - `match/structure/`
  - `match/transitions/`
  - `match/flow/`

2. Regles de scoring
- comment un joueur progresse ou est penalise
- couche principale :
  - `match/scoring/`

3. Conditions de victoire
- quand la partie se termine
- couche principale :
  - `match/victory/`

4. Regles fines de runtime
- comportements metier optionnels ou parametrables
- couche principale :
  - `config/fine_rules_config.py`
  - `match/flow/trick_rules.py`
- actuellement :
  - `Uniqueness`
  - `Repetition`

5. Regles de dictionnaire
- qu'est-ce qu'un trick valide
- quelles constructions sont autorisees
- quelle est l'identite canonique d'un trick
- couche principale :
  - `dictionary/`

6. Regles de discipline
- specialisations liees a un sport/profil
- actuellement :
  - `inline / inline_primary_grind`
- couche principale :
  - `dictionary/`

Familles futures probables

Regles d'attaque plus riches
- `Advantage`
- `Clash`
- `Redo`

Regles de restriction de dictionnaire
- `Banned Tricks`
- limitation de familles de tricks
- limitation de longueur de combo

Regles de scoring alternatives
- points
- difference
- temps / rounds combines a la victory

Regles de discipline plus larges
- autres dictionnaires inline
- skate
- bmx

Point a retenir
Le moteur ne doit plus etre pense comme un simple jeu de lettres parametrable.
Apres V8, il est deja structure pour recevoir :
- des regles de match
- des regles fines
- des familles de scoring/victory
- des dictionnaires de discipline
