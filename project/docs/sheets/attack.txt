Name
Attack

Type
parameter

Ensemble
Attack & Defense

Parent Rule
None

Definition
Parameter that defines how many attempts the attacker may use to impose a trick during one attack turn.

Purpose
Define how many attack tries the attacker may spend on the trick engaged during the current attack turn.

Options
Value (option): defines the number of attack attempts granted to the attacker.
- Default Value: 1, standard default value
- Other common values: 2 or 3
- Choice: allows free numeric input, within the product limit

Activation Conditions
None

Dependencies
active attack turn

Interactions
Multiple Attack
Repetition
Advantage
Clash

Default Behavior
The attacker receives a fixed number of attempts defined before the match.

Advanced Behavior
If Attack is greater than 1, advanced behaviors such as Multiple Attack may become available.
If Attack is greater than 1 and Multiple Attack remains Disabled with No Repetition Off, Attack may also create a compatibility constraint with Repetition.

Code Impact
attack-attempt resolution
turn flow
action validation

UX Notes
Must be presented as a simple numeric parameter.
Recommended values may be shown as shortcuts, but they still represent one single numeric choice.

Open Questions
Some advanced rule combinations may restrict which Attack / Repetition pairs are valid.
