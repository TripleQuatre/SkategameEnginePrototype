Documentation V8 - Orchestration du moteur

Important
Le projet n'est plus centre sur un dossier `engine/` comme dans les versions
plus anciennes. En V8, l'orchestration du moteur est partagee entre :

- `application/`
- `controllers/`
- `match/`

Role global
L'orchestration active d'une partie est maintenant assuree par :

- `application/game_session.py`
- `controllers/game_controller.py`
- `match/flow/turn_flow.py`

`application/game_session.py`
- facade applicative principale
- construit le runtime a partir de `MatchConfig`
- coordonne :
  - structure
  - flow
  - validation
  - persistence
  - undo
  - transitions de roster
  - dictionnaire de tricks
- expose les operations de haut niveau :
  - `start_game`
  - `start_turn`
  - `resolve_attack`
  - `resolve_defense`
  - `cancel_turn`
  - `undo`
  - `save_game`
  - `load_game`
  - suggestions/resolution de tricks V8

`controllers/game_controller.py`
- relais tres fin vers `GameSession`
- API stable pour CLI, GUI et tests

`match/flow/turn_flow.py`
- coeur de l'orchestration d'un tour
- compose :
  - `AttackFlow`
  - `DefenseFlow`
  - `TurnCycle`
  - `TrickRules`
  - scoring runtime
  - victory runtime

Pieces importantes du flow

`match/attack/attack_flow.py`
- demarre un tour
- gere les tentatives d'attaque
- decide si l'attaque continue, passe en defense, ou echoue

`match/defense/defense_flow.py`
- gere la reponse des defenseurs
- pilote la progression defenseur par defenseur
- delegue la resolution fine a `DefenseAttemptResolver`

`match/flow/turn_cycle.py`
- consomme le trick courant
- avance au prochain attaquant
- termine proprement un tour ou une partie
- historise les echec d'attaque utiles a `Repetition`

`match/flow/turn_state.py`
- fonctions de mutation d'etat de bas niveau
- ouverture/fermeture du tour
- promotion attaque -> defense
- nettoyage runtime

Ce qu'il faut retenir
- l'orchestration n'est plus concentree dans un seul fichier
- `GameSession` est la facade applicative
- `TurnFlow` est le coeur runtime d'un tour
- `match/` est le vrai centre moteur actuel
